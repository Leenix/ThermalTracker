Constructor & Destructor Documentation

TrackedBlob::TrackedBlob ( )

Create a new tracked blob. Track blobs start off empty until they are initialised by a regular blob using the set method. After that, you can update the blob that is being tracked to record its movements between frames (and thus updates)
